PRIMITIVOS
Se accede directamente al valor declarado, y dichos valores no se alterarán

  • String
  • number
  • boolean
  • null
  • undefined
  • nan

COMPUESTOS
Se accede a la referencia del valor (ejempo ajedres: caballo a5 hace referencia al lugar al que debe estar el caballo ni al caballo en si mismo.). Permite que agreguemos o quitemos elementos lo que hará que el valor final pueda cambiar.

  • object={}
  • array=[]
  • function(){}
  • Class{}
  • undefined

CADENAS DE TEXTO

Una cadena de texto es lo que se encuentra encerrado entre comillas dobles o simples "" - ''
