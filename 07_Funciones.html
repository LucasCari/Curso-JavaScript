Funciones: function(){};

Una funcion es un bloque de código autocontenido {}, que se puede definir una vez y ejecutar en cualquier momento. Opcionalmente, una funcion puede aceptar parámetros y devolver un valor.
Las funciones en JS son objetos, un tipo especial de objetos.
Pueden asignarse a un valor, y pueden pasarse como argumento y usarse como un valor de retorno.

  • Funciones declaradas:
    • function returnValue(){
    console.log('uno');
    }
    Para que una funcion funcione debemos invocarla. [newFunction();]

    Puede invocarse en cualquier parte de nuestro código, incluso antes de que la funcion sea declarada. Cuando la invocamos antes de la declaracion se elvará como sucede con las variables globales.

  • Funciones Expresadas:
    • const functionExpresada = () => {
    console.log('esto es una funcion expresada, es decir una funcion que se la ha asignado como valor a una variable, si invocamos esta funcion, antes de su definición JS nos dirá.... "Uncaught ReferenceError: Cannot access "functionExpresada" before initialization"')
    }
    functionExpresada();

  • return:
    • function returnValue(){
    console.log('uno');
    return 19;
    console.log('dos');
    console.log('tres');
    return 'la función ha retornado una Cadena de Texto';
    }

    Solo retornará hasta [ return 19; ], ya que al encontrar la pablara return deja de procesar el resto
