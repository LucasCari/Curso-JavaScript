Objects

object = objetos { }

En JS todo es un objeto [consola:: __proto__ ]
Es ideal trabajarlos mediante CONST, para evitar que modifiquemos su valor.
Un objeto es una coleccion de llaves valores

Atributos/Propiedades de un objeto

Los atributos son las variables/ propiedades que se encuentran dentro de un objeto.

Funciones (metodos)

A las funciones se les llama métodos. Todas los metodos dentro de un objeto son funciones y todas llevan ()
