Operadores Relacionales

Son los que nos permiten comparar (<) mayor que, (>) menor que, (>=) mayor igual, (<=) menor o igual, (==) igual igual, (===) triple igual, (!=) diferente de, (!==).
(=) : Asignación lo que esta de lado derecho se lo estoy asignando al lado derecho (a = 5) el valor de a es 5
(==) : Comparacion de valores: Comparamos si el valor de A = B, sin depender el tipo de caracter (0 == false [true, ya que ambos tienden a ser falsos]) // ('7' == 7 [true: ya que tanto la cadena de texto como el numero tienen un valor de 7])
(===) : Comparacion de tipo de datos y valores: Es lo ideal, ya que comparamos tanto tipo de datos como su valor (7 === '7' [False: ya que no son el mismo tipo de datos, aunque si tienen el mismo valor asignado.])
(0 == false [false, ya que ambos tienden a ser falsos, pero uno es numero y otro es boolean])

Operadores de incremento y decremento

Nos permitiran incrementar o disminuir el valor de las variables que nosotros deseemos. Tambien se puede multiplicar, dividir, o cualquier operador aritmetico.
Para subir los calores por unidad podemos expresarlo ++i (i++ )o --i (i--)

Operadores Lógicos

- Not [!] (no)::Niega, lo que es verdadero lo vuelve falso y viceversa
- Or [||] (ó):: ((9===9)||('9'===9))[TRUE] Cuando tenga 2 o mas condisiones, con que una sola sea verdadera todo el resultado será
- And [&&] (y..):: ((9===9)&&('9'===9))[FALSE] la condisión && necesita que ambas condisiones sean verdaderas para validarlo, de caso contrario será falso
