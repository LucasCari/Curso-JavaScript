Estructuras de Control

Permite controlar el flujo de nuestra programación. Nos permitirán tomar ciertas acciones o no; dependiendo de la evaluacion de una condision ((9===9)||(9==='9))

  • Secuenciales:: una detras de la otra
  • Estructuras condicionales: dependden de si se cumple o no una condisión
  • Loops: Repetititvas

Condicionales

IF{}else{} [IF{X}] Si se cumple la condición ejecutara "X", [ELSE{Y}] Pero sino cumplirá "Y"

IF{}else{} anidados::

if(hora>=0&&hora<=5){
console.log("Dejame dormir");
}else if(hora>=6&&hora<=11){
console.log('Buenos días');
}else if(hora>=12&&hora<=18){
console.log('Buenas tardes');
}else{
console.log('Buenas noches')
}

Operador Ternario::

Solo se puede ejecutar una sola línea de código, es decir que sirve como un IF.else pero en una sola linea. En el caso de tener mas de una condision se deberá utilizar if else anidados.
(condición) ? verdadero:flaso
let eresMayor=(edad>=18)?'Eres mayor de edad':'eres menor de edad';

switch-case::

Sirve para cuando tenemos que evaluar diferentes valores para una misma variable
switch (key) { [Key= variable]
case 0: [Variable a analizar (0); Doble punto es { apertura de llave]
console.log('Domingo');
break; [ } llave de cierre del analisis incial ]
default: [En el caso que las condiciones que informamos no existan]
console.log('El dia no existe');
break;
}
